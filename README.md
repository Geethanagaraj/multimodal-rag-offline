# multimodal-rag-offline
Multimodal Offline RAG system that ingests PDFs, DOCX, images, and audio, performs unified semantic search using embeddings, and generates grounded answers with citation transparency using a local LLM.

## Running the App

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   streamlit run app.py

3. Open:
   http://localhost:8501
